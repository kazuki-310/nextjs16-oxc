# Todo リスト画面

## 画面概要

- ページ見出し「Todo リスト」と、追加用フォーム、Todo の一覧を表示する。
- 追加時は `useHookFormOptimisticAction` により、サーバー応答前に一覧先頭へ仮の行を挿入する。

## 表示項目

| 大項目   | 中項目   | コンポーネント | 概要                                    |
| -------- | -------- | -------------- | --------------------------------------- |
| 見出し   |          | Title          | `Todo リスト`                           |
| フォーム | タイトル | TextInput      | プレースホルダー「新しい Todo を入力」  |
|          | 追加     | Button         | 送信中 loading                          |
| 一覧     | 各行     | Checkbox       | `todo.title` をラベルに表示、`readOnly` |

## 空状態

- 楽観的状態が0件のとき「データがありません」。
