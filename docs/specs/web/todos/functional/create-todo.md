# Todo 作成

## 概要

- `createTodo` サーバーアクション（`authClient`）で Todo を作成する。
- 成功後 `revalidatePath("/todos")` する。

## リクエスト

| パラメータ名 | 型     | 必須 | 制約      |
| ------------ | ------ | ---- | --------- |
| title        | string | 必須 | 1文字以上 |

## 処理内容

- セッションの `user.id` を `authorId` として `prisma.todo.create` する。

## エラーハンドリング

| 条件        | 扱い                                                    |
| ----------- | ------------------------------------------------------- |
| DB 等の失敗 | `Error`「作成処理に失敗しました」（アダプタ経由で通知） |
