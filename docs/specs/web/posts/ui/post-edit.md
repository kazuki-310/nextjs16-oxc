# 投稿編集画面

## 画面概要

- パスパラメータ `id` の投稿を読み込み、タイトル・本文を編集して更新する。
- データ読み込みは `Suspense` 内のサーバーコンポーネントで行う。

## 表示項目

| 大項目   | 中項目   | コンポーネント | 概要             |
| -------- | -------- | -------------- | ---------------- |
| 見出し   |          | Title          | `投稿編集`       |
| フォーム | id       | hidden input   | 更新対象 ID      |
|          | タイトル | TextInput      | 既存値を初期表示 |
|          | 本文     | Textarea       | 既存値を初期表示 |
|          | 更新     | Button         | 送信中 loading   |

## バリデーション

- `updatePostSchema`: `id`・タイトル・本文（新規作成と同様の文字数制約）。

## 権限・注意

- 画面の表示は「ログイン済み」であれば可能（取得は ID のみ）。
- 更新実行時のみ `authorId` とセッションが一致するか検証され、不一致ならエラーとなる。

## 404

- 存在しない `id` の場合、サーバー側で `notFound()` となり 404 ページになる。
