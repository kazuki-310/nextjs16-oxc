# 投稿新規作成画面

## 画面概要

- タイトルと本文を入力し、サーバーアクションで投稿を作成する。

## 表示項目

| 大項目   | 中項目   | コンポーネント | 概要                  |
| -------- | -------- | -------------- | --------------------- |
| 見出し   |          | Title          | `投稿作成`            |
| フォーム | タイトル | TextInput      | プレースホルダーあり  |
|          | 本文     | Textarea       | 自動伸縮、最低5行程度 |
|          | 作成     | Button         | 送信中 loading        |

## バリデーション

- `createPostSchema`: タイトル必須・最大100文字、本文必須。

## 権限

- 認証済みユーザー。作成者 ID はセッションから付与される。
