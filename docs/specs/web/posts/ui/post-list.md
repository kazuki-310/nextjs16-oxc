# 投稿一覧画面

## 画面概要

- 投稿をテーブルで一覧表示する。
- 右上（見出し行）から新規作成ページ（`/posts/new`）へ遷移できる。
- データ取得は `Suspense` 内のサーバーコンポーネントで行う。

## 表示項目

| 大項目   | 中項目   | コンポーネント | 概要                             |
| -------- | -------- | -------------- | -------------------------------- |
| 見出し   |          | Title          | `投稿一覧`                       |
| 操作     | 新規     | Link           | `/posts/new` へ「新規作成」      |
| テーブル | タイトル | Table.Td       | 投稿タイトル                     |
|          | 投稿者   | Table.Td       | `author.name`                    |
|          | 作成日   | Table.Td       | `createdAt` を ja-JP で表示      |
|          | 操作     | Anchor         | 各行「編集」→ `/posts/{id}/edit` |

## 空状態

- 投稿が0件のとき「データがありません」（dimmed テキスト）。

## 権限

- 認証済みユーザー（`(auth)` レイアウト配下）。
