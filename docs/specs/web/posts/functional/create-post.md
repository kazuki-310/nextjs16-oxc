# 投稿作成

## 概要

- `createPost` サーバーアクション（`authClient`）で投稿を作成する。

## リクエスト

| パラメータ名 | 型     | 必須 | 制約                 |
| ------------ | ------ | ---- | -------------------- |
| title        | string | 必須 | 1文字以上100文字以下 |
| content      | string | 必須 | 1文字以上            |

## 処理内容

- セッションの `user.id` を `authorId` として `prisma.post.create` する。
- 成功後 `/posts` へ `redirect` する。

## エラーハンドリング

| 条件           | 扱い                                    |
| -------------- | --------------------------------------- |
| DB 等の失敗    | `ActionError`「作成処理に失敗しました」 |
| バリデーション | フォームにスキーマメッセージを表示する  |
