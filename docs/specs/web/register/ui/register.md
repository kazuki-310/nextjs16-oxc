# 従業員登録画面

## 画面概要

- 新規従業員の identifier（ニックネームまたはメール）とパスワードを入力し、登録する。
- ログイン画面と同様のフィールド構成で、見出しは「従業員登録」、送信ボタンは「登録」。

## 表示項目

| 大項目   | 中項目                             | 小項目 | コンポーネント     | 概要                  |
| -------- | ---------------------------------- | ------ | ------------------ | --------------------- |
| 見出し   |                                    |        | プレーンテキスト   | `従業員登録`          |
| フォーム |                                    |        |                    |                       |
|          | ニックネーム または メールアドレス |        | テキストインプット | 必須（`authSchema`）  |
|          | パスワード                         |        | パスワード入力     | 必須（`authSchema`）  |
|          | 登録                               |        | ボタン             | 送信中は loading 状態 |

## バリデーション

- `authSchema` に準拠（ログイン画面と同一）。

## 権限

- 未認証ユーザーがアクセスする想定（ルーティングはアプリ構成に従う）。
