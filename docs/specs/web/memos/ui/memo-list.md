# メモ帳画面

## 画面概要

- ページ見出し「メモ帳」。
- 上部に新規メモフォーム（タイトル・本文・保存）。
- 下部に自分のメモをカード一覧表示。各カードに更新日時と削除ボタンがある。

## 表示項目

| 大項目   | 中項目   | コンポーネント | 概要                            |
| -------- | -------- | -------------- | ------------------------------- |
| 見出し   |          | Title          | `メモ帳`                        |
| フォーム | タイトル | TextInput      | 必須                            |
|          | 本文     | Textarea       | 必須、複数行                    |
|          | 保存     | Button         | 送信中 loading                  |
| 一覧     | タイトル | Text           | `memo.title`                    |
|          | 本文     | Text           | `memo.body`（改行保持）         |
|          | 更新日時 | Text           | `memo.updatedAt` をロケール表示 |
|          | 削除     | Button         | 確認ダイアログなしで削除実行    |

## 空状態

- メモ0件のとき「メモはまだありません」（dimmed）。

## 権限

- 認証済みのみ。未認証でコンテナがマウントされた場合は `/login` へリダイレクトする。

## ナビゲーション

- サイドバーに「メモ帳」リンク（`/memos`）を置く。
