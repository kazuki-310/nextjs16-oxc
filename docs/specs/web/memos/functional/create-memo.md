# メモ作成

## 概要

- `createMemo` サーバーアクション（`authClient`）で `memos` テーブルに1件追加する。

## リクエスト

| パラメータ名 | 型     | 必須 | 制約        |
| ------------ | ------ | ---- | ----------- |
| title        | string | 必須 | 1〜100文字  |
| body         | string | 必須 | 1〜5000文字 |

## 処理内容

- セッションの `user.id` を `authorId` として `prisma.memo.create` する。
- 成功後 `revalidatePath("/memos")` する。

## エラーハンドリング

| 条件           | 扱い                                      |
| -------------- | ----------------------------------------- |
| DB 等の失敗    | `ActionError`「メモの作成に失敗しました」 |
| バリデーション | フォームにスキーマメッセージを表示する    |
