# メモ削除

## 概要

- `deleteMemo` サーバーアクション（`authClient`）でメモを1件削除する。

## リクエスト

| パラメータ名 | 型     | 必須 | 概要    |
| ------------ | ------ | ---- | ------- |
| id           | number | 必須 | メモ ID |

## 処理内容

1. `id` で `memo` を取得する。存在しなければ `ActionError`「メモが見つかりません」。
2. `memo.authorId` とセッションの `user.id` が一致しない場合、`ActionError`「削除権限がありません」。
3. `prisma.memo.delete` を実行する。
4. `revalidatePath("/memos")` する。

## エラーハンドリング

| 条件        | 扱い                                      |
| ----------- | ----------------------------------------- |
| DB 等の失敗 | `ActionError`「メモの削除に失敗しました」 |
