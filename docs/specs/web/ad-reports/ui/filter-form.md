# 広告レポート・フィルタフォーム

## 画面概要

- キャンペーン名の部分一致、最小インプレッション数、最小 CV 数で一覧行を絞り込む。
- 「検索」で URL クエリに反映、「リセット」で初期値に戻す。

## 表示項目

| ラベル                 | コンポーネント | 概要                       |
| ---------------------- | -------------- | -------------------------- |
| キャンペーン名         | TextInput      | ローカル state、送信で反映 |
| 最小インプレッション数 | NumberInput    | min 0                      |
| 最小CV数               | NumberInput    | min 0                      |
| 検索                   | Button         | `setParams` でクエリ更新   |
| リセット               | Button         | 入力とクエリを初期化       |

## フィルタロジック（クライアント）

- `name`: 行の `name` に対し大文字小文字を区別しない部分一致。
- `minImpressions` > 0 のとき `row.impressions` が閾値以上。
- `minConversions` > 0 のとき `row.conversions` が閾値以上。
