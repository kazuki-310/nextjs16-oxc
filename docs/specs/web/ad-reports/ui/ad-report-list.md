# 広告レポート一覧画面（通常）

## 画面概要

- ページタイトル「広告レポート一覧」。
- フィルタフォーム、列表示スイッチ、複数のデータテーブルを縦に並べる。
- テーブルデータ取得中は `TablesLoadingFallback` を表示する。

## 表示項目（ページ構成）

| 大項目         | 概要                                                |
| -------------- | --------------------------------------------------- |
| 見出し         | `広告レポート一覧`（`Title` order={1}）             |
| フィルタ       | [フィルタフォーム](./filter-form.md) と同様         |
| 列表示         | [列表示コントロール](./column-visibility.md) と同様 |
| データテーブル | 設定ごとに `AdDataTable`。ソート可能な列見出しあり  |

## クエリパラメータ（nuqs）

- フィルタ: `name`, `minImpressions`, `minConversions`（詳細は `lib/schema` の `filterParsers`）。
- 非表示列: `hidden`（文字列の配列）。

## 権限

- 認証済みユーザー（`(auth)` 配下）。
