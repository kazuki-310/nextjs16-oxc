# 広告レポート・列表示コントロール

## 画面概要

- `columnDefs` に定義された各列について、スイッチで表示／非表示を切り替える。
- 状態は URL クエリ `hidden`（非表示にする列キーの配列）に保持する。

## 表示項目

| 要素      | 概要                                       |
| --------- | ------------------------------------------ |
| 各 Switch | ラベルは列の日本語名（例: キャンペーン名） |
| checked   | `hidden` にキーが含まれていない＝表示      |

## テーブル連携

- テーブル側では `hidden` を `VisibilityState` に変換し、TanStack Table の列表示に反映する。
- テーブルヘッダーから列を隠した場合も `onColumnVisibilityChange` で `hidden` クエリを更新する。
