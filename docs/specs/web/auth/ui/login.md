# ログイン画面

## 画面概要

- ニックネームまたはメールアドレスとパスワードを入力してログインする。
- 送信時はサーバーアクションを実行し、失敗時は通知（トースト）でエラーを表示する。

## 表示項目

| 大項目   | 中項目                             | 小項目 | コンポーネント     | 概要                  | デザイン |
| -------- | ---------------------------------- | ------ | ------------------ | --------------------- | -------- |
| 見出し   |                                    |        | プレーンテキスト   | `ログイン`            |          |
| フォーム |                                    |        |                    |                       |          |
|          | ニックネーム または メールアドレス |        | テキストインプット | 必須（スキーマ準拠）  |          |
|          | パスワード                         |        | パスワード入力     | 必須（8〜32文字など） |          |
|          | ログイン                           |        | ボタン             | 送信中は loading 状態 |          |

## バリデーション（クライアント）

`authSchema` に準拠する（例: メール形式、ニックネームの許容文字、パスワード長）。

## 権限

- 未認証ユーザーがアクセスする。
