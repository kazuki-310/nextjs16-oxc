# 集中ヒント画面

## 画面概要

- 認証レイアウト配下のルート `/focus-tips` を表示する。
- ページタイトルは「集中ヒント」とする。

## 表示項目

| 大項目 | 中項目       | 小項目 | 概要                                                                               |
| ------ | ------------ | ------ | ---------------------------------------------------------------------------------- |
| ヘッダ | タイトル     |        | 「集中ヒント」                                                                     |
| 本文   | 基準日       |        | ヒント選定に使った日付（YYYY-MM-DD、サーバー基準）                                 |
| 本文   | ヒント本文   |        | `focus_tips` を sort_order 昇順で読み、日付剰余で 1 件。0 件時はフォールバック文言 |
| 補足   | 説明テキスト |        | ヒントが日替わりであること、本文が DB の focus_tips 由来であることを短文で示す     |

## 振る舞い

- ヒント候補は `focus_tips.body` の一覧とする（件数 0 のときは画面上でフォールバック文言を表示する）。
- 表示するインデックスは `基準日` の数値（例: `YYYYMMDD` を整数化）を候補数で割った余りで決める（処理詳細は `functional/fetch-daily-focus-tip.md`）。

## 権限

- 認証が必要なルートグループ内に配置されている（ミドルウェア・レイアウトの設定に従う）。
