# 日次集中ヒントの取得

## 概要

`focus_tips` テーブルからヒント候補を読み取り、サーバー日付に応じて 1 件を選んで返す。

## 処理

1. `sort_order` 昇順で `focus_tips` を全件取得する。
2. 件数が 0 のときは、フォールバック用の固定文言を返す（画面が空にならないようにする）。
3. 件数が 1 以上のとき、基準日を `YYYYMMDD` 相当の整数にした値を件数で割った余りをインデックスとして、並び順上の該当行の `body` を返す。

## エラー

- DB 接続エラーは呼び出し側（ページ）に委ね、必要ならエラーバウンダリや再試行ポリシーで扱う（本仕様では詳細は定めない）。
